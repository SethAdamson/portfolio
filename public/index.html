On-Target

On-Target was a personally created web application that was designed using Trello as a style guide to create a KanBan board application

  • Live Site
  • GitHub

Photos

Description

I set out to create a Trello-like application that would have full drag and drop functionality, with the ability to handle multiple users and their previously setup boards. Completing a Full-Stack-Application on my own for the first time was challenging and invigorating.

Technology

React | Redux | Node | MassiveJS | Express | PostgreSQL | React-DnD | Auth0 | AmazonS3 | NodeMailer

  • React-DnD (Drag and Drop) was the most difficult piece of technology on this project. This was due to having to self teach implementation with only the documentation.
  • React & Redux were used to handle this Front-End display of data. Data transfer was a major part of the project, thus making Redux integral.
  • All Front-End server communication was handled within Reducer. This was used to handle the transfer of data, while manipulating Front-End display for increased user experience speed.
  • Customized Auth0 login. Live NodeMailer for user communication.
  • AmazonS3 used for media storage and recall.
